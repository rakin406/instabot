# instabot
Instagram chatbot for my personal use.

Requirements:

1. Python
2. Brave/Chrome/Chromium browser

## Installation
```shell
git clone https://github.com/rakin406/instabot.git
cd instabot
pip3 install -r requirements.txt
```

## Usage
```shell
./bot.py <instagram>
./bot.py _chemeraku_
```

Login to Instagram and restart the program.
